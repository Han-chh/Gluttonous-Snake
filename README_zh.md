# 贪吃蛇游戏

使用Swing GUI框架实现的经典贪吃蛇游戏。

## 描述

这是一个流行的贪吃蛇游戏的简单实现，玩家控制一条蛇通过吃苹果来增长，同时避免与墙壁和自身身体碰撞。游戏具有图形界面，具有实时更新、计分和计时器功能。

## 功能

- **经典游戏玩法**：使用键盘输入控制蛇（A左转，D右转）
- **动态蛇增长**：蛇根据长度改变颜色（青色→绿色→蓝色→黄色→粉色→灰色→黑色）
- **实时计分**：跟踪蛇长度和经过时间
- **苹果生成**：红色苹果每5秒随机出现
- **碰撞检测**：当蛇撞到墙壁或自身身体时游戏结束
- **用户界面**：干净的GUI，信息面板显示当前统计和控制按钮

## 要求

- Java运行环境（JRE）8或更高版本

## 运行

```bash
java -jar Gluttonous-Snake.jar
```

## 游戏控制

- **A**：蛇左转
- **D**：蛇右转
- **START按钮**：开始游戏
- **INSTRUCTIONS按钮**：查看游戏规则

## 游戏规则

1. 这是一场饥饿蛇的游戏
2. 使用A或D让蛇左转和右转
3. 尝试吃红色苹果让自己变长！
4. 永远不要撞到边界或自己的身体

## 项目结构

```
src/game/
├── Game.java          # 主类和入口点
├── Window.java        # 主窗口和GUI设置
├── Snake_head.java    # 蛇逻辑和移动
├── Control.java       # 游戏控制和地图初始化
├── Map_things.java    # 单个地图网格单元
├── Apple.java         # 苹果（食物）类
├── Decorations.java   # 装饰基类
├── ButtonPerform.java # 按钮动作处理器
├── timer.java         # 游戏计时的计时器线程
```
